# CareerRecipe
This website helps student connect with professionals all over the world working in their preferable career field

Features:
  1. Anonymous Roadmap Sharing
  2. Share career journeys anonymously
  3. Discuss successes, failures, and lessons learned
  4. Help future students make informed decisions

  1. Artificial Intelligence
  2. Robotics
  3. Software Engineering
  4. Medicine
  5. Cybersecurity
  6. Data Science
  7. Entrepreneurship
